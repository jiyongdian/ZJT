- **分层解耦架构**：采用 `api/` (路由层) -> `services/` & `agents/` (业务逻辑与智能体层) -> `model/` (数据访问层) 的经典三层结构，并通过 `perseids_server/client.py` 桥接认证与算力微服务。
- **动态驱动工厂模式**：`task/pipeline_drivers` 与 `visual_drivers` 将数十种外部 AI 服务（如 RunningHub、Kling）封装为统一接口，由 `PipelineProcessor` 进行异步提交与状态轮询。
- **统一配置与热更新**：`unified_config.py` 建立“数据库优先 + YAML 兜底”的注册表机制，支持任务类型、驱动实现及算力规则的运行时热更新。
- **全链路算力风控**：在 API 网关、智能体工具调用及任务执行器等关键节点集成 `check_computing_power` 校验，确保资源消耗受控。
- **混合前端编排引擎**：结合 Vue 3 状态管理与原生 Canvas 高性能渲染，通过 `node_registry.js` 实现工作流节点的动态扩展与持久化。
