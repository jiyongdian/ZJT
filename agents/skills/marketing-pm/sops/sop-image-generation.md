# 营销图片生成流程

## 适用场景
用户需要生成或编辑营销图片，包括但不限于：
- 商品展示图 / 产品图
- 广告海报 / 宣传图
- 社交媒体配图（小红书、抖音封面、朋友圈等）
- 品牌视觉素材
- **图片编辑**：基于用户提供的原始图片进行风格转换、背景替换、元素修改等

## 流程步骤

### 步骤1：收集需求信息

通过 `ask_user` 工具向用户确认以下信息（如果用户描述中已包含则跳过）：

1. **图片用途**：商品展示？海报？社交媒体配图？图片编辑修改？
2. **是否有原始图片**：用户是否提供了需要编辑的原始图片URL？
3. **主体内容**：展示什么商品/场景/人物？编辑指令是什么？
3. **风格偏好**：写实？插画？3D渲染？扁平化？
4. **尺寸比例**：1:1（Instagram）、4:3（小红书）、16:9（横版海报）、9:16（竖版/手机）
5. **其他要求**：颜色倾向、氛围、参考图等

**示例**：
```
ask_user(
    question="请确认图片用途和风格偏好：",
    options=["商品白底图", "商品场景图", "社交媒体配图", "广告海报"]
)
```

### 步骤2：委托给图片专家

收集完需求后，调用 `call_agent` 将任务委托给营销图片专家。

**文生图场景**（用户没有提供原始图片）：
```
call_agent(
    AgentName="marketing-image",
    task_description="请为用户生成一张营销图片，需求如下：
    - 用途：[图片用途]
    - 主体：[主体内容]
    - 风格：[风格偏好]
    - 比例：[尺寸比例]
    - 其他要求：[额外要求]

    请先检查模型和算力，然后构建提示词并生成图片。",
    user_input="[用户的原始描述]"
)
```

**图片编辑场景**（用户提供了原始图片URL）：
```
call_agent(
    AgentName="marketing-image",
    task_description="请为用户编辑图片，需求如下：
    - 原始图片URL：[图片URL]
    - 编辑指令：[用户希望如何修改图片]
    - 比例：[尺寸比例]

    请使用 edit_image 工具，传入 image_url 参数和编辑指令。",
    user_input="[用户的原始描述]"
)
```

### 步骤3：等待专家返回结果

图片专家会返回生成结果，包含：
- 图片 URL（如果成功）
- 使用的模型名称
- 消耗的算力
- 或者失败原因

### 步骤4：向用户报告结果

将专家返回的结果转达给用户：
- **成功**：告知图片 URL 和消耗的算力
- **处理中**：告知用户图片正在生成，稍后可通过对话再次查询
- **失败**：告知失败原因，询问是否需要重试或换一种方式

**示例回复**：
```
图片已生成完成！
- 模型：GPT Image 2
- 消耗算力：XX
- 图片链接：[URL]

如需调整，请告诉我您希望修改的方面。
```

### 步骤5：迭代优化（如用户需要）

如果用户对结果不满意：
1. 收集用户的修改意见
2. 再次调用 `call_agent` 生成新图片，在 `task_description` 中包含修改要求
3. 重复步骤3-4

## ⚠️ 注意事项

1. **不要自己调用生图工具**：生图操作全部委托给 `marketing-image` 专家
2. **需求收集要充分**：确保在委托前了解用户的完整需求
3. **算力提醒**：如果专家报告算力不足，需要明确告知用户
4. **不要轮询**：图片专家内部会处理状态查询，PM Agent 不需要重复查询
