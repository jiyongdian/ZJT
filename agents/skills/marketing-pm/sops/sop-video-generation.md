# 营销视频生成流程（商业版专属）

## 适用场景
用户需要生成营销视频，包括但不限于：
- 商品展示视频
- 广告宣传视频
- 社交媒体短视频
- 品牌宣传片

## 流程步骤

### 步骤1：告知用户此功能为商业版专属

**立即**使用 `ask_user` 工具向用户说明：

```
ask_user(
    question="视频生成功能仅商业版本支持，请购买商业版本付费解锁。如需了解更多，请联系客服或访问官网。",
    options=["了解其他功能", "联系客服"]
)
```

### 步骤2：引导用户

根据用户回复：
- 如果用户选择"了解其他功能"：引导用户使用图片生成等其他免费功能
- 如果用户选择"联系客服"：提供联系方式或官网链接
- 如果用户表达了其他需求：帮助用户使用当前可用的功能

## 注意事项

1. **不要尝试调用视频生成工具**：此功能在社区版中不可用，即使尝试调用也会失败
2. **礼貌引导**：不要简单拒绝，而是引导用户了解商业版的价值
3. **不要自己调用任何视频相关工具**：社区版不包含视频生成能力
